互动式强化学习教程蒙特卡洛方法：从数据中学习

这是一套可以在网页里直接把玩的强化学习教程。它从蒙特卡洛（Monte Carlo, MC）方法讲起—— 这是你遇到的第一类免模型（model-free）算法：不需要知道环境的转移规律， 只靠和环境互动攒下的「经验」就能学会做事。

每一章都包含三类内容：正文讲解（生动直觉 + 必要数学）、互动实验（紫色面板，拖滑块、点按钮，亲手观察算法行为）、 编程实践（深蓝色面板，提示你在终端运行哪些代码、改哪些参数、观察什么变化）。 建议按顺序阅读，每遇到一个互动面板都先动手玩一玩、再继续往下读。

第 1 章

没有地图怎么办：蒙特卡洛预测

从抛硬币和大数定律出发，理解「采样求平均」如何替代模型；亲手在网格世界里估计一个策略的状态价值 V(s)。
第 2 章

从评估到改进：MC Basic 与 Exploring Starts

把策略迭代中的「模型」抽掉，得到最朴素的 MC 控制算法；探究回合长度与稀疏奖励，学会把一条轨迹「榨干」。
第 3 章

告别上帝视角：MC ε-Greedy

去掉「探索性出发」这一不现实的假设，用软性策略保证探索；亲手体会探索与利用 (exploration vs. exploitation) 的根本权衡。

使用方法

网页部分

纯静态页面，无任何外部依赖。直接用浏览器打开即可；也可以起一个本地服务：

# 任选其一
xdg-open ~/studyspace/RL/mc-tutorial/index.html
# 或
cd ~/studyspace/RL/mc-tutorial && python3 -m http.server 8000
# 然后访问 http://localhost:8000

编程实践部分

实践脚本在 practice/ 目录下（改编自 Reinforcement_learning_tutorial_with_demo 仓库的 notebook，整理为可直接运行、带实验开关的脚本）。只需 numpy 和 matplotlib：

cd ~/studyspace/RL/mc-tutorial/practice
python3 01_mc_prediction.py      # 第1章：MC 预测
python3 02_mc_es.py              # 第2章：MC Exploring Starts
python3 03_mc_epsilon_greedy.py  # 第3章：MC ε-Greedy

每个脚本顶部都有一块「实验区」常量，正文里的实践环节会提示你改哪些、看什么。

素材来源与进一步阅读

  • 主线脉络与 5×5 网格世界实验，取自赵世钰《强化学习的数学原理》第 5 章 Monte Carlo Methods （本目录 Book-Mathematical-Foundation-of-Reinforcement-Learning/，含 PDF、课件和官方网格世界 Python 环境）。
  • 实践代码改编自 Reinforcement_learning_tutorial_with_demo 仓库的 monte_carlo_demo / monte_carlo_es_demo / monte_carlo_epsilon_greedy_demo 三个 notebook。
  • 直觉讲法部分参考《Easy RL：强化学习教程》第 3 章「免模型预测与控制」。

学完三章后，自然的下一步是时序差分（TD）方法——它解决了 MC「必须等回合结束才能学」的痛点，对应原书第 7 章与 demo 仓库的 td_prediction.ipynb / SARSA_demo.ipynb。
